**🟢 Good fit — auto-route when enabled:**
- Bug fixes with clear reproduction steps

- Documentation fixes and README updates

- Architecture decisions (route to Danny)
- UI/UX and design decisions (route to Basher)
- Cross-component interactive features (route to Rusty)
- QA and acceptance criteria verification (route to Livingston)
